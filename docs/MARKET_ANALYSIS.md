# Market Analysis: Indonesian Tech & Creative Landscape (2024-2025)

## 1. Overview

The Indonesian digital market is maturing rapidly. The demand has shifted from "just having a website" to "integrated digital ecosystems" that include mobile apps, ERPs, and AI capabilities. This analysis covers typical pricing, services, and tech stacks found among top-tier and mid-tier Software Houses and Digital Agencies in Indonesia (Jakarta, Bandung, Bali, Yogyakarta).

## 2. Competitor Archetypes & Pricing

### A. The "Digital Agency" (Marketing & SEO Focus)

- **Focus**: Brand awareness, traffic, social media management, basic web presence.
- **Services**: SEO, SEM (Google Ads), Social Media Management (SMM), WordPress Development.
- **Tech Stack**: WordPress, PHP, Elementor, Yoast SEO, Google Analytics, Meta Pixel.
- **Pricing (IDR)**:
  - **SEO/AIO**: 5,000,000 - 15,000,000 / month (Retainer).
  - **Landing Page**: 3,000,000 - 10,000,000 (One-off).
  - **Company Profile**: 8,000,000 - 20,000,000.

### B. The "Software House" (Custom Dev Focus)

- **Focus**: Business logic, automation, custom applications, startups.
- **Services**: Web Apps, Mobile Apps (iOS/Android), ERP/CRM, API Development.
- **Tech Stack**:
  - _Frontend_: React.js, Next.js, Vue.js.
  - _Backend_: Node.js, Go (Golang), Laravel (PHP), Python (Django/FastAPI).
  - _Mobile_: Flutter, React Native, Swift, Kotlin.
  - _Infra_: AWS, Google Cloud, Docker, Kubernetes.
- **Pricing (IDR)**:
  - **Custom Web App (MVP)**: 35,000,000 - 80,000,000.
  - **Mobile App (Hybrid)**: 50,000,000 - 120,000,000.
  - **Enterprise ERP/CRM**: 150,000,000 - 500,000,000+.
  - **Man-Month Rate**: 15,000,000 - 30,000,000 per developer.

### C. The "AI & Data Consultant" (Niche/Emerging)

- **Focus**: Machine Learning, Chatbots, Data Analysis, Automation.
- **Services**: Custom LLM integration, RAG (Retrieval-Augmented Generation), Predictive Analytics.
- **Tech Stack**: Python, TensorFlow, PyTorch, OpenAI API, LangChain, Pinecone, Hugging Face.
- **Pricing (IDR)**:
  - **AI Chatbot Integration**: 20,000,000 - 50,000,000.
  - **Custom AI Model/Fine-tuning**: 100,000,000+.
  - **Consultation**: 2,000,000 - 5,000,000 / hour.
